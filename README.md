# VoyagerSplit
A wireless low profile keyboard made for [Hack Club](https://hackclub.com/)'s [Blueprint](https://blueprint.hackclub.com/) program. It uses a staggered columnar layout with 28 keys on each side, with 4 thumb buttons. It uses Kailh Choc switches and Seeed Studio Xiao BLE microcontrollers.

## Why?
I decided to design this keyboard to help improve my typing speed and to give me a mechanical keyboard option for when I'm travelling or away from my PC, as I go into college. I wanted to keep it fairly similar to a conventional keyboard though to prevent too much learning curve.
